# Medical Platform - React & Django

Une **plateforme médicale web** développée avec **React** pour le frontend et **Django** pour le backend.  
Elle permet la gestion des utilisateurs (patients, médecins, pharmaciens), des ordonnances et du stock de médicaments, avec des fonctionnalités spécifiques selon le rôle.

---

##  Technologies utilisées

- **Frontend :** React, JavaScript, HTML, CSS, Tailwind CSS   
- **Backend :** Django, Python, SQLite  
- **Gestion de versions :** Git & GitHub  

---

## Rôles et fonctionnalités

### Patient
- Consulter ses ordonnances créées par le médecin  
- Rechercher la disponibilité des médicaments dans les pharmacies  
- Interface claire et responsive  

### Médecin
- Créer et gérer les ordonnances pour les patients  
- Voir la liste des ordonnances qu’il a créées avec les détails des patients et médicaments  

### Pharmacien
- Gérer le stock de médicaments  
- Les prix des médicaments sont récupérés automatiquement depuis la base de données  
- Voir les ordonnances pour vérifier la disponibilité des médicaments
